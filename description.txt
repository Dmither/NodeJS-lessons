Node.js - платформа для запуску JS-коду поза браузером

Підготовка до роботи:
 1. Для роботи необхідно скачати встановити NodeJS (LTS), 
    разом з ним встановиться пакетний менеджер (NPM)
 2. Перевірити, чи встановлений коректно (версію) 
    [node --version], [npm --version]
 3. Ініціалізувати проєкт [npm init]

За допомогою NPM можна встановлювати та видаляти пакети
 1. Встановити [npm i []]
 2. Видалити [npm uninstall []]
 3. Встановити всі пакети [npm i]
При встановленні пакетів інформація прописуються в package.json
Зберігати можна проєкт без node_modules та package-lock.json

Використання бібліотек (вбудованих чи встановлених) в скриптах:
 1. Підключити бібліотеку [const os = require('os')]

Створення та підключення власного модуля:
 1. Створити файл/папку
 2. Прописуємо відповідні поля/методи
 3. Вказуємо дані для експорту (module.exports)
    module.exports = {
      addition: addition,         // із стрілкової функції
      subtraction: subtraction,   // із звичайної функції
      multiplication: (a, b) => { // стрілкова функція
        return a * b;
      },
      division: (a, b) => {
        return a * b;
      },
    }
 4. Підключаємо в проєкт (const my_pack = require('./my_pack'))

Робота з файлами та папками:
 1. Підключаємо модуль 'fs' (file system)
 2. Зчитати з файла [fs.readFileSync(file, code)] (code='utf-8')
 3. Запис до файлу [fs.writeFileSync(file, data)]
     (створить неіснуючий, перезапише текст існуючого)
 4. Видалити файл [fs.unlinkSync(obj)]
 5. Створити папку [fs.mkdirSync(dir)]
 5. Видалити папку [fs.rmdirSync(dir)] (порожню)
 (відповідно readFile, writeFile, unlink, mkdir, rmdir
  працюють асинхронно, при їх виконанні програма не зупиняється; 
  мають додатковий аргумент - функція-колбек при завершенні роботи
  fs.writeFile(file, data, (error, data) => {}))

Команди терміналу:
  clear - очистка вмісту
  echo - вивести текст (echo \"Error: no test\")
  && - об'єднання команд
  node [file] - запуск файла за допомогою Node.js 
    (шлях має бути вказаний відносно поточної директорії)
  npm run [script] - виконання скрипта npm (з package.scripts)
  npm i/install [] - встановити щось за допомогою npm
  npm uninstall [] - видалити щось за допомогою npm
Прапори терміналу:
  -g, --global - встановити глобально (на ПК)
  -D, --save-dev - встановити лише для розробки  
Навігація:
  ./ - поточна директорія
  ../ - верхня директорія
