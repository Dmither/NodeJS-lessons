Бібліотека Express призначена для зручної роботи з веб-серверами

Створення серверу:
 1. Встановити і підключити бібліотеку:
    [npm i express], const express = require('express');
 2. Ініціалізувати змінну сервера: const app = express();
 3. Вказати адресу: app.get('/', (req, res) => { ... });
    (через ":" для динамічної адреси: '/user/:username/:userid')
    (вивести текстову інформацію: res.send('This is home page');)
 4. Ініціалізувати сервер: app.listen(port, () => { ... })
