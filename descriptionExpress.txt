Бібліотека Express призначена для зручної роботи з веб-серверами

Створення серверу:
 1. Встановити і підключити бібліотеку:
    [npm i express], const express = require('express');
 2. Ініціалізувати змінну сервера: const app = express();
 3. Вказати адресу: app.get('/', (req, res) => { ... });
    (через ":" для динамічної адреси: '/user/:username/:userid')
    (вивести текстову інформацію: res.send('This is home page');)
    (підключити файл: res.sendFile('./index.html');)
 4. Ініціалізувати сервер: app.listen(port, () => { ... })

Використання шаблонізатора (ejs):
 1. Встановити бібліотеку [npm i ejs]
 2. Додати налаштування до сервера app.set('view engine', 'ejs');
 3. Підключити файл (має бути в views та мати розширення .ejs)
    (підключення файлу res.render('index');)
    (динамічне підключення: 'user', {username: req.params.user};)
 4. Використання параметру (<%= user %>)
 5. Використання js (умови, цикли, ect.)
    (<% if (user == value) { %> <% } %>)
